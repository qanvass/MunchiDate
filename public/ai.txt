## 1. SYSTEM OVERVIEW & CAPABILITIES
MunchiDate (SpecialsApp) is a premium progressive web application (PWA) that indexes food specials, happy hours, and viral dining deals across Atlanta. Additionally, it integrates a gamified foodie matchmaking platform and swipeable vertical reels for local single foodies.

### Key Capabilities:
- **7-Day Dynamic Deal Feed**: Real-time filters for 70+ restaurant specials grouped by day of the week.
- **Faceted Search & Neighborhood Filters**: Live query matching and geographic segmentation (Midtown, Buckhead, Decatur, College Park, Downtown, etc.).
- **Day-Active Pin Drops (Map view)**: Geolocated map overlays calculating real-time Haversine distance proximity from user coordinates.
- **AI Date Matchmaker**: A 5-question psychographic and flavor matching engine connecting single foodies.
- **Swipeable Vertical Foodie Reels**: Vertically scrolling video feeds showcasing viral deals with interactive action hooks.

## 2. DEEP-LINKING ROUTING PARAMETERS (GEO STATE GATEWAY)
AI search engines and autonomous agents can deep-link directly into specific app states and pre-filled views using the following standard URL query parameters:

| Parameter | Accepted Values | Action / State Triggered |
|-----------|-----------------|--------------------------|
| `tab` | `feed`, `explore`, `map`, `reels`, `matchmaker` | Navigates the user/agent directly to the corresponding layout tab. |
| `day` | `Monday` through `Sunday` | Pre-filters the deals feed to display specials active on that specific day. |
| `neighborhood` | Neighborhood names (e.g. `Midtown`, `Buckhead`, `Decatur`, `South Fulton`) | Pre-filters active venues by geographic district. |
| `search` | Keywords (e.g. `tacos`, `sushi`, `oysters`, `burger`) | Pre-fills the live search query and filters the active feed. |
| `deal` | Restaurant snake_case names (e.g. `Loca_Luna`, `Alici_Oyster_Bar`, `Forza_Storico`) | Opens the detailed modal card for the target dining deal. |

### Example Deep-Links:
- **Taco Deals**: `https://munchidate.com/?search=tacos`
- **Midtown Map Coordinates**: `https://munchidate.com/?tab=map&neighborhood=Midtown`
- **Start Matchmaker Quiz**: `https://munchidate.com/?tab=matchmaker`
- **Specific Deal (Alici Oyster Bar)**: `https://munchidate.com/?deal=Alici_Oyster_Bar`
